# JSPM | GULP | BABEL Boilerplate project

A Gulp, JSPM and Babel boilerplate project

## Installation

#### 1. Install jspm CLI:

```
    npm install jspm -g
```

#### 2. Initialise jspm project:

```
    jspm init
```

#### 3. Install Gulp globally:

```
	npm install gulp -g
```

#### 3. Install gulp plugins from devDependencies:

```
	npm install
```

#### 4. Run gulp:

```
	gulp
```
